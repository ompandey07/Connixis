Features

Powerful Tools for Network Engineers

Everything you need to design, configure, and simulate professional network infrastructures.

Drag & Drop Builder

Intuitive visual interface with 50+ device types. Simply drag devices and connect them with multiple cable types.

Cisco-like CLI

Practice with realistic command line interface. Execute show commands, configure routing, VLANs, and ACLs.

Packet Simulation

Visualize data flow with animated packets. Run ping tests, traceroutes, and see traffic in real-time.

Security Config

Configure firewalls, access control lists, NAT rules, and VLANs. Learn network security hands-on.

Routing Protocols

Configure static routes, RIP, OSPF, EIGRP, and BGP. Understand how routing protocols work.

Save & Export

Save topologies as JSON files. Export network diagrams as images. Share configurations easily.
